# Weather Diary — Дневник погоды

**Автор:** Юнкус Матвей

## Описание программы

Weather Diary — это графическое приложение для ведения дневника погоды. Позволяет добавлять записи о погоде (дата, температура, описание, наличие осадков), фильтровать их по дате и температуре, а также сохранять и загружать данные в формате JSON.

## Функциональные возможности

- Добавление записей с проверкой корректности ввода
- Отображение записей в таблице
- Фильтрация по дате (точное совпадение)
- Фильтрация по температуре (выше указанного значения)
- Сохранение всех записей в JSON-файл
- Загрузка записей из JSON-файла
- Автоматическая загрузка данных при запуске
